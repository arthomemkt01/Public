1. เปิด Google Sheets → Extensions → Apps Script

2. วางโค้ดจากไฟล์ arthome_apps_script.gs แล้วกด Save

3. กด Deploy → New deployment → Web app

4. Copy URL ที่ได้ แล้วแก้ไขในไฟล์ HTML บรรทัด const SHEET_API = 'YOUR_APPS_SCRIPT_URL_HERE'
